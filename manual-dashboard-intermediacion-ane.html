Preguntas frecuentes

Errores comunes del dashboard.

FAQ Centro de Control

❔ ¿Puedo cargar solo un archivo?

El diseño está pensado para tres fuentes. Si falta una, algunos indicadores pueden quedar incompletos.

❔ ¿Qué pasa si un archivo tiene columnas diferentes?

El sistema puede no reconocer algunos campos. Revise encabezados y formato antes de cargar.

❔ ¿Qué significa un duplicado?

Puede ser una misma persona, cita o registro repetido. Debe validarse antes de eliminar o interpretar.

❔ ¿Los datos se envían a internet?

En esta versión, el procesamiento se realiza en navegador. Las librerías pueden cargarse desde CDN si el archivo lo requiere.

❔ ¿Cómo lo uso para una reunión?

Cargue datos actualizados, revise resumen, embudo, equipo y validación. Luego imprima o comparta pantalla.
