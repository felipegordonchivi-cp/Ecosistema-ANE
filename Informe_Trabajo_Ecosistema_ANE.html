Módulos del Ecosistema

El proyecto Ecosistema ANE integra 4 módulos especializados de reclutamiento y orientación laboral:

1. ATS ANE - Motor de Preselección Curricular

Descripción: Sistema de ranking curricular automático con competencias ESCO-CR

Mejoras realizadas:

  • Reescritura de CSS móvil para iPhone con breakpoints completos
  • Optimización de responsive design en ATS
  • Corrección de visibilidad de contenido en dispositivos móviles

2. Simulador STAR - Entrevistas Virtuales por Competencias

Descripción: Plataforma de entrevistas con audio, video y transcripción en tiempo real

Mejoras realizadas:

  • Implementación de WebRTC para bidireccionalidad de audio y video
  • Corrección de race condition en ICE candidates
  • Solución de pantalla en blanco al admitir candidatos
  • Sincronización de transcripción candidato-entrevistador
  • Integración de Firebase para gestión de salas
  • Estilo Teams: cámara y micrófono siempre activos
  • Mejora de responsive design y CSS móvil en iPad/iPhone

3. Centro de Control - Dashboard Intermediación Laboral

Descripción: Dashboard ejecutivo con analytics de Chart.js + reportes XLSX

Mejoras realizadas:

  • Mantenimiento de funcionalidad de exportación de datos
  • Corrección de CSS para visualización en múltiples dispositivos
  • Optimización de fallback CDN (cdnjs → jsDelivr → unpkg)

4. Laboratorio IA - Copiloto ESCO-CR

Descripción: Motor de IA con recomendación responsable y calibración de confianza

Mejoras realizadas:

  • Reescritura completa de CSS móvil para Laboratorio IA
  • Corrección de IntersectionObserver en dispositivos iOS
  • Garantía de visibilidad de contenido .reveal en móvil
  • Optimización de responsive design en múltiples tamaños de pantalla

Características Técnicas Implementadas

Stack Tecnológico

  • HTML5 + CSS3 (Responsive + Mobile-First)
  • JavaScript Vanilla (sin dependencias externas)
  • WebRTC (RTCPeerConnection, getUserMedia)
  • Firebase Realtime Database (para sincronización)
  • MediaRecorder API (para transcripción)
  • Chart.js (para analytics)
  • IndexedDB / LocalStorage (almacenamiento local)
  • Service Workers (para offline-first)

Mejoras de Arquitectura

  • CSS Grid y Flexbox para layouts responsive
  • Media queries: mobile (320px), tablet (768px), desktop (1024px+)
  • Procesamiento 100% local (sin datos en servidores externos)
  • Privacy-by-design: CVs y datos nunca salen del navegador
  • Fallback CDN inteligente para resilencia
  • CSP (Content Security Policy) compatible
